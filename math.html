Derivative

Calculus
The rate at which a function changes at any given point. It represents the slope of the tangent line to the function at that point.
Formula:
f'(x) = lim(h→0) [f(x+h) - f(x)] / h
Example:
The derivative of f(x) = x² is f'(x) = 2x
calculusrate of changeslope

Integral

Calculus
The reverse operation of differentiation. It represents the area under a curve or the accumulation of quantities.
Formula:
∫ f(x) dx
Example:
∫ x² dx = x³/3 + C
calculusareaantiderivative

Quadratic Formula

Algebra
A formula that provides the solutions to any quadratic equation of the form ax² + bx + c = 0.
Formula:
x = (-b ± √(b² - 4ac)) / (2a)
Example:
For x² - 5x + 6 = 0: x = (5 ± √(25-24)) / 2 = (5 ± 1) / 2
algebraquadraticformula

Pythagorean Theorem

Geometry
In a right triangle, the square of the hypotenuse equals the sum of squares of the other two sides.
Formula:
a² + b² = c²
Example:
In a triangle with sides 3, 4, and 5: 3² + 4² = 9 + 16 = 25 = 5²
geometrytriangletheorem

Sine Function

Trigonometry
A trigonometric function that gives the ratio of the opposite side to the hypotenuse in a right triangle.
Formula:
sin(θ) = opposite / hypotenuse
Example:
sin(30°) = 1/2
trigonometryfunctionratio

Natural Logarithm

Algebra
The logarithm to the base e (approximately 2.718). It is the inverse function of the exponential function.
Formula:
ln(x) = log_e(x)
Example:
ln(e) = 1, ln(e²) = 2
logarithmexponentialfunction

Standard Deviation

Statistics
A measure of the amount of variation or dispersion in a set of values.
Formula:
σ = √(Σ(x - μ)² / N)
Example:
For data set [2, 4, 4, 4, 5, 5, 7, 9], σ ≈ 2.0

Limit

Calculus
The value that a function approaches as the input approaches some value.
Formula:
lim(x→a) f(x) = L
Example:
lim(x→0) (sin x / x) = 1
calculuslimitapproach

Matrix

Linear Algebra
A rectangular array of numbers, symbols, or expressions arranged in rows and columns.
Formula:
A = [aᵢⱼ] where i is row, j is column
Example:
A 2×2 matrix: [[1, 2], [3, 4]]

Binomial Theorem

Algebra
A formula for expanding powers of binomials.
Formula:
(x + y)ⁿ = Σ(k=0 to n) C(n,k) × xⁿ⁻ᵏ × yᵏ
Example:
(x + y)² = x² + 2xy + y²
