Standard Deviation

Statistics
A measure of the amount of variation or dispersion in a set of values.
Formula:
σ = √(Σ(x - μ)² / N)
Example:
For data set [2, 4, 4, 4, 5, 5, 7, 9], σ ≈ 2.0
